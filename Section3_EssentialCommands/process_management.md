# Process Management Commands

- `ps aux`: Lists all running processes with detailed info.
- `top`: Interactive real-time process monitoring.
- `htop`: Advanced interactive process viewer (needs installation).
- `kill <PID>`: Terminates a process using its Process ID.
- `pkill <process_name>`: Kill processes by their command name.
- `nice -n <priority> <command>`: Run commands with specified scheduling priority.
